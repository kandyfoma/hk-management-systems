## 🚢 Deployment Instructions

### Step 1: Prepare
```bash
cd frontend/website
npm install
```

### Step 2: Test Locally
```bash
npm run build
npm run start
```
Visit http://localhost:3000 to verify

### Step 3: Deploy to Vercel
**Option A: Using Vercel CLI**
```bash
npm install -g vercel
vercel --prod
```

**Option B: Using GitHub**
1. Push to GitHub
2. Create project on vercel.com
3. Set root directory to `frontend/website`
4. Deploy

### Step 4: Configure Domain
1. Add domain www.katohms.com in Vercel
2. Update DNS records at registrar
3. Wait 24-48 hours for propagation

### Step 5: Verify
1. Visit https://www.katohms.com
2. Test all links
3. Check mobile responsiveness
4. Verify app.katohms.com redirect
